树莓派 GPIO 调试 TB6600 步进电机驱动器

2026-07-14
#raspberry-pi  #gpio  #stepper-motor  #硬件 

起因

手头有个小项目需要精确控制电机转动角度，于是买了一块 TB6600 步进电机驱动器和一颗 42 步进电机。控制器用的是树莓派 4B，通过 GPIO 直接发脉冲。本以为照着淘宝店家的接线图接上就能转，结果折腾了整整一个下午。

TB6600 简介

TB6600 是东芝出品的一款两相步进电机驱动芯片，市面上常见的模块把芯片、光耦隔离、拨码开关都集成在了一块板子上。主要特点：

  • 供电电压 9~42V DC
  • 最大 4A 电流输出
  • 支持 1/2/4/8/16 细分
  • 共阳 / 共阴接法都支持

六个拨码开关：S1S3 设电流，S4S6 设细分。控制端有 ENA+/-（使能）、DIR+/-（方向）、PUL+/-（脉冲）三组信号。

接线

树莓派 40pin GPIO 是 3.3V 电平，TB6600 的控制信号输入端内部有光耦，推荐用共阳极接法：

树莓派 5V  (Pin 2/4)   →  ENA+ / DIR+ / PUL+
树莓派 GPIO23 (Pin 16) →  PUL-  (脉冲)
树莓派 GPIO24 (Pin 18) →  DIR-  (方向)
树莓派 GPIO25 (Pin 22) →  ENA-  (使能)

关键点：TB6600 的控制信号触发的是下降沿。光耦的阳极接 5V，阴极（PUL-）被 GPIO 拉低时导通，产生一个有效脉冲。所以 GPIO 默认保持高电平，拉低再拉高才算一个脉冲。
