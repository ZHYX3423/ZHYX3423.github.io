重新开始写博客

今天花了一些时间把 Hugo 站点重新搭起来了。选择 Hugo 的原因很简单——快，真的很快。

一个静态站点生成器，用 Go 写成，编译一个中等规模的站点只需要几百毫秒。对比以前用过的某些工具动辄几十秒的构建时间，Hugo 的体验确实让人愉悦。

为什么选 Terminal 主题

这次用了 Terminal 主题，风格极简，没有多余的装饰。黑底白字，等宽字体，就像在终端里阅读一样。对于以文字为主的个人博客来说，这种设计恰到好处——让读者专注于内容本身。

  • 写几篇关于日常开发踩坑的记录
  • 也许可以分享一下 Windows 上的一些效率工具

总之，又一个开始。希望这次能坚持下去。
